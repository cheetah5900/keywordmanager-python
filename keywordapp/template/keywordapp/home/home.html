เพิ่มหน้า

1. กรอกหน้า เช่น "สมัครสมาชิก"
2. กรอก Title ของหน้า เช่น "ทำอาหาร" และกดปุ่มเพิ่มข้อมูล

เพิ่ม Meta Description

3. กดปุ่ม Meta Description เพื่อเข้าหน้าจัดการ Meta Description
4. กรอก Keyword แบบเต็ม (ไม่ต้องตัดคำซ้ำ ใส่คำเต็มได้เลย) เช่น "ทำอาหาร กลางวัน"
5. ในส่วน "รวมคำ" จำรวมคำแบบเต็ม และ แบบย่อ ไว้ ถัดลงมาจะเป็น การตัดคำซ้ำให้อัตโนมัติ 2 แบบ คือ

5.1 ไม่ซ้ำกันเลย : แต่ละคำ ซ้ำไม่เกิน 1ด้วย Spacebar

5.2 ซ้ำกันไม่เกิน 2 ครั้ง : แต่ละคำ ซ้ำไม่เกิน 1ด้วย Spacebar และซ้ำไม่เกิน 2 ครั้ง

6. สามารถกด "ใช้คำชุดนี้" เพื่อเอาคำชุดนี้ไปเป็น Keyword แบบย่อได้

8. ใส่ Keyword ที่ต้องการให้เป็น Header จากนั้นเลือกประเภทของ Header (H1-H6) จากนั้นกดเพิ่ม
